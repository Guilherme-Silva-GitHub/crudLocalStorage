# JavaScript CRUD

This project is a simple CRUD (Create, Read, Update, Delete) application built with JavaScript, CSS, and HTML. It leverages the browser's localStorage feature to store data locally. 
With this application, users can perform basic CRUD operations on data without the need for a server-side database.

Users can seamlessly manage data with this application, making it ideal for small-scale projects or learning purposes.
